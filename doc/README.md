# 📚 FastAPI AI Agent 平台教程

> 零基础入门 FastAPI + 大语言模型开发实战课程

## 🎯 课程简介

本教程将带领你从零开始，掌握现代 Python Web 开发技术栈。通过搭建一个完整的 **AI Agent 对话平台**，你将学习到：

- 🚀 **FastAPI 框架** - 现代、高性能的 Python Web 框架
- 🧠 **大语言模型集成** - 调用通义千问/OpenAI API
- 📊 **数据库操作** - SQLAlchemy ORM 异步操作
- 🏗️ **分层架构设计** - Router → Service → Repository 模式
- 🔄 **异步编程** - Python async/await 最佳实践

## 🚀 快速开始

```bash
# 1. 创建虚拟环境
python -m venv venv
source venv/bin/activate  # macOS/Linux

# 2. 安装依赖
cd project
pip install -r requirements.txt

# 3. 配置环境变量
cp .env.example .env
# 编辑 .env 填入 LLM API Key

# 4. 启动应用
uvicorn app.main:app --reload

# 5. 访问 http://localhost:8000
```
